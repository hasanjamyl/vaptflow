# Sample Cybersecurity Scan Reports

This folder contains sample cybersecurity scan reports for testing VaptFlow.

Users can download these files and upload them into VaptFlow to test:

- Document preview
- Vulnerability extraction
- Remediation planning
- Compliance mapping
- Executive summary
- Technical summary
- AI-assisted report Q&A

## Important Notice

These files are for educational and testing purposes only.

Do not upload real client reports, real production IP addresses, credentials, internal URLs, hostnames, or confidential security information.
